# README – Concierto de Los 40 Pop

## Descripción del proyecto
Este proyecto consiste en una página web sencilla hecha con Bootstrap. La web muestra información sobre el “Concierto de Los 40 Pop”, incluyendo qué es el evento, la fecha, el lugar y una programación.

## Secciones de la web y columnas usadas
La página está dividida en:

- **Navbar**: con los enlaces a información y contacto.
- **Hero**: título del evento, una descripción, la fecha y una imagen.
- **Sección de información**: está hecha con una fila y dos columnas (col-md-6 y col-md-6).
- **Footer**: incluye el contacto y el correo del evento.

## Componentes de Bootstrap utilizados
- Navbar
- Clases de espaciado (py-5, mb-4…)
- Clases de color (bg-dark, bg-light, text-white…)
- Botón del menú responsive
