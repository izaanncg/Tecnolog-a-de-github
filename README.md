# Tecnolog-a-de-github
Descripción del proyecto
Este proyecto es una página web sencilla hecha con Bootstrap 5. El objetivo era crear una web informativa sobre el “Concierto de Los 40 Pop”, mostrando la información básica del evento: qué es, dónde se hace, la fecha y una pequeña programación.
Secciones de la web y columnas usadas
-Navbar: incluye los enlaces a información y contacto.
-Hero: aquí aparece el título del evento, una descripción corta, la fecha y una imagen de la Plaza del Pilar.
-Sección de información: está hecha con una fila (row) y dos columnas (col-md-6 y col-md-6).
-Footer: incluye el contacto y el correo del evento
